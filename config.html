Sensitivity
This option increases the sensitivity of the selected audio source. A higher value will make the graphic to react even to low sounds, while lower value will decrease the sensitivity to loud sounds
Smoothing
Smoothing indicates how fluid the graphic will be. A high value increases the framerate to a maximum value of 60 frames per second, while a lower value will decrease such framerate. the minimum value is 1 as 1 frame per second
Bit Sample
Bit Sample indicates the amount of details the graphic will have. A high value increases the value of details, while a lower value will decrease details. Be aware that certain visualizations that uses a high value could increment the use of CPU, so use it carefully.
