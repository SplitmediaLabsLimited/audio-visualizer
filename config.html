Sensitivity
Increasing this value will make
the visualizer respond to quieter sounds
and make the graph bigger.
Smooth Delay
Higher values adds a delay to the visualization which makes it look much smoother and calmer.
Smooth Wave
Higher values makes the wave to display more evenly.
Bit Sample
A number value for the frequency resolution of the output data. Increasing this value will add more CPU load. Graphic will be reloaded upon change.
