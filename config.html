Sensitivity
This option increases the sensitivity of the selected audio source. A higher value will make the graphic to react better to low sound frequencies, while lower value will decrease the sensitivity to loud sounds frequencies.
Smoothing
Smoothing indicates how fluid the graphic will be. leaving smoothing as 0 will make the graphics react quite quickly, while leaving it to 0.99 will decrease the reaction, but also will smooth the changes on the data.
Bit Sample
Bit Sample indicates the amount of details the graphic will have. A high value increases the value of details, while a lower value will decrease details. Be aware that certain visualizations that uses a high value could increment the use of CPU, so use it carefully.

Bar Count
Depending on the selected bitrate, the maximum allowed value may change, for example: if you want 20 columns in an bit sample of 32 bits, the maximum amount will be 16 columns. Keep in mind to divide by half the bitrate to know your maximum amount of bars.
Space
by default, there is a space of one pixel between columns. you can increase this value to experiment the look and feel of the visualization.
